# 🌍 Wanderlust Travel Guide
A full-stack travel guide app to explore destinations, chat with an AI travel bot, and save favourites. Built with HTML, CSS, JS, Node.js, Express &amp; MongoDB, with OpenAI integration.

## ✨ Features

- 🔎 Browse beautifully described travel destinations
- 💬 Chat with an AI Travel Bot powered by OpenAI GPT
- ⭐ Save destinations to your favourites
- ❌ Delete destinations from your favourites list
- 📬 Submit queries via contact form (stored in MongoDB)
- 🧠 Server-side validation and error handling
- 📁 Static frontend + Express backend
- 🔐 API key secured via `.env`

---

## 📁 Tech Stack

| Layer       | Technology             |
|-------------|-------------------------|
| Frontend    | HTML, CSS, JavaScript,Tailwind CSS|
| Backend     | Node.js, Express.js     |
| Database    | MongoDB + Mongoose      |
| AI API      | OpenAI (GPT-3.5 Turbo)  |
| Form Valid. | express-validator       |
